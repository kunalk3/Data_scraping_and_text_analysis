## :wrench: Installation
- Create __virtual environment__ `python -m venv VIRTUAL_ENV_NAME` and activate it `.\VIRTUAL_ENV_NAME\Scripts\activate`.
- Install necessary library for this project from the file `requirements.txt` or manually install by `pip`.
  ```
  pip install -r requirements.txt
  ```
  To create project library requirements, use below command,
  ```
  pip freeze > requirements.txt
  ```
  
- _Selenium webdriver:_ WebDriver is an open source tool for automated testing of webapps across many browsers. It provides capabilities for navigating to web pages, user input, JavaScript execution, and more. package is used to automate web browser interaction from Python. Several browsers/drivers are supported (Firefox, Chrome, Internet Explorer), as well as the Remote protocol. Use the package manager [pip](https://pypi.org/project/selenium/) to install selenium.

Download Chrome Driver from [here](https://chromedriver.chromium.org/). Place downloaded driver at location _C:\Program Files (x86)\Google\chromedriver_ OR if changed, need to update the path in code.
  
---
  
- Now execute the python notebook.
  ``` 
  Naukri Web Scraping.ipynb
  ```
